# Remax Wechat Typescript Demo

[remax](https://github.com/remaxjs/remax)

## Command

**Install**

```bash
$ yarn install
```

**Develop**

```bash
$ yarn dev
```

Then open `dist` filefolder in wechat developer tool.

**Build & Publish**

```bash
$ yarn build
```

## Test features

* typescript / wxapp API
* react hooks / function component
* scss / linter
